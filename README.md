# 🚀 Professional Portfolio | Suleman Saqib

A high-end, futuristic professional portfolio website designed for **Suleman Saqib**, a BSIT student at Air University Islamabad, **Cyber Security Enthusiast**, and **Full Stack Developer**.

![Portfolio Preview](assets/images/Profile.jpeg)

## 🌟 Key Features

- **Futuristic UI/UX**: Dark mode theme with neon cyan accents, glassmorphism, and cinematic animations.
- **Dynamic Roles**: Typing animation showcasing expertise in **Cyber Security**, **Full Stack Development**, and **WordPress**.
- **Interactive Certificates**: Integrated PDF viewer for 8 professional certificates & licenses.
- **Cyber Security Focus**: Detailed expertise in **Kali Linux**, **Nmap**, and **Wireshark**.
- **Project Showcase**: Detailed project cards with direct links to GitHub repositories and LinkedIn demonstration posts.
- **Technical Skills Matrix**: Categorized expertise in Programming, Backend Development (Node.js/MongoDB/Oracle), and Network Security.
- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile devices.

## 🛠️ Built With

- **Core**: HTML5, CSS3, Vanilla JavaScript (coded in **VS Code**)
- **Design System**: Glassmorphism, CSS Variables, Flexbox/Grid
- **Icons**: FontAwesome 6
- **Typography**: Google Fonts (Inter, Space Grotesk)
