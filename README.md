# 🌐 Personal Portfolio Website — Suleman Saqib

A fully responsive, modern personal portfolio website built with pure HTML, CSS, and JavaScript — featuring cinematic animations, interactive UI elements, and a clean professional design.

🔗 **Live Site:** [sulemansaqib243342.github.io/Portfolio](https://Sulemansaqib243342.github.io/Portfolio/)

---

## ✨ Features

- 🎨 **Premium Dark UI** — Glassmorphism design with smooth gradient accents
- ⚡ **Animated Loader & Transitions** — Cinematic page entry experience
- 🖱️ **Custom Cursor** — Interactive cursor with follower effect
- ✍️ **Typing Animation** — Dynamic role display in the hero section
- 🌌 **Particle Canvas Background** — Animated particle system built with vanilla JS
- 📱 **Fully Responsive** — Optimized for all screen sizes and devices
- 📜 **Scroll Progress Bar** — Visual reading progress indicator
- 🗂️ **Sections Covered:**
  - Hero, About, Skills, Services
  - Projects (with GitHub + LinkedIn links)
  - Experience Timeline
  - Certificates & Licences (PDF viewer modal)
  - Contact Form

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure & semantic markup |
| CSS3 | Styling, animations, glassmorphism |
| Vanilla JavaScript | Interactivity, canvas particles, typing effect |
| Font Awesome 6.5 | Icons throughout the site |
| Google Fonts (Inter + Space Grotesk) | Typography |

> No frameworks. No libraries. Pure HTML/CSS/JS.

## 🌍 Deployment

Deployed via **GitHub Pages** — automatically served from the `main` branch.

To deploy your own fork:
1. Fork this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch → `/root`
4. Your site will be live at `https://<your-username>.github.io/Portfolio/`
